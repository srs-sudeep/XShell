# Bundled Plugin Commands

The current checkout ships with these bundled plugin command groups:

## Git plugin

```bash
gst
glog
gdiff
gadd
gcommit
gpush
gpull
gbranch
gcheckout
gstash
grebase
```

## Sysinfo plugin

```bash
sysinfo
diskusage
netinfo
procs
uptime
```

## Calc plugin

```bash
calc
convert
hex2dec
dec2hex
dec2bin
```

For full descriptions from the running shell:

```bash
plugin info git
plugin info sysinfo
plugin info calc
```
